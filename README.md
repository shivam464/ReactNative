Q1- what is reconciliation?
Ans:-
    In React Native, reconciliation is the process React uses to efficiently update the user interface (UI) on the screen to match changes in component state or props .

Q2- what is Diffing ?
Ans:- 
    Diffing: The new virtual tree is compared with the previous version using React's diffing algorithm. This algorithm identifies exactly what has changed, added, or been removed in the UI tree.
